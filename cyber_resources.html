Cyber Security Resources

This page contains resources I recommend, on the topic of cyber security.

Websites

  • The register. Great security news website, if you can get over the snarky language.
  • The Hacker News.

Podcasts

  • Security Now. I have been listening to this podcast since I started university in 2007.
  • Risky Biz. Great breaif 20min updates each week, on the latest news. I tend to skip the rest.
  • Darknet Diaries. Great stories. I recommend starting with the 2 part ‘xbox underground’ episodes.

Books

  • Building secure & reliable systems at google.A great book about security architecture at scale.
  • The phoenix project and the sequel The unicorn project.. If you work in technology, this is a must read. It's also just an interesting novel, rather than a traditional technology text book.
  • Google site reliability engineering.This book describes how google approaches the task of managing there IT infrastructure.
  • A bug hunters Diary. About a security researcher, and how he finds security vulnerabilities.
  • The art of exploitation. A book that talks you through deep level exploitation of computer software. A little out of date now, but a great place to start with memory corruption.
